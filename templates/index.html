Hello!

This application connects to the GitHub API and gets the most starred repository for these 5 languages:

  • JavaScript
  • Python
  • Java
  • Ruby
  • C

This first feature is triggered by the blue button above. But, that's not all this app does.

This app also keeps record of these repos, and you can check the history by clicking on the green button.

If you want more details on each repo, just click the blue buttons on the Name of Repo column and you'll get access to a description and also to the repo's URL.
